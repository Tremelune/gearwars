# gearwars
A shitty gearing calculator

# setup
Install Node.js and NPM (`brew install node`), then:

`$ npm install`

# run tests
`$ npm test`

# run local
`$ npm start`

# debug local
First launch the app:

`$ npm start`

Then run the "Chrome" debug tast in VS Code. It's defined in launch.json

# production release
`$ npm run deploy`
